# 用户协议

最后更新日期：2026 年 7 月

## 一、服务说明

南评 Nanping（以下简称"本平台"）是一个面向南京大学学生的课程评价信息平台。用户可以在本平台查看、发布课程评价，为选课决策提供参考。

本平台为非官方、非营利性质的第三方工具，与南京大学不存在隶属、授权或合作关系。

## 二、账号注册与使用

1. 用户须使用南京大学邮箱（@nju.edu.cn 或 @smail.nju.edu.cn）注册账号。
2. 用户应对其账号下的一切行为负责，不得将账号出借、转让给他人使用。
3. 用户可申请注销账号。法律法规另有规定或为履行平台义务所必需保存的信息除外，平台将在合理期限内删除或匿名化处理相关个人信息。
4. 如发现账号存在异常登录或违规行为，本平台有权暂停或终止该账号的使用权限。

## 三、用户行为规范

用户在使用本平台时，不得有下列行为：

1. 发布违法信息，包括但不限于危害国家安全、破坏民族团结、宣扬暴力恐怖等内容。
2. 发布侮辱、诽谤、人身攻击、歧视性或其他侵害他人合法权益的信息。
3. 发布与课程评价无关的广告、垃圾信息或恶意链接。
4. 使用自动化程序批量提交评价或刷分。
5. 利用技术手段干扰本平台正常运行，包括但不限于爬虫攻击、SQL 注入、DDoS 攻击等。

## 四、评价内容规范

1. 评价应客观、真实地反映课程的实际情况，包括授课内容、考核方式、给分标准、教师教学水平等。
2. 用户应基于真实经历发布评价，并保证其发布内容真实、合法。如因虚假评价产生法律责任，由发布者自行承担。
3. 评价中如包含引述他人观点或材料的，应注明出处。
4. 本平台鼓励真实、具体、具有参考价值的课程评价，不鼓励情绪化宣泄、恶意刷评、重复评价或其他影响社区质量的内容。
5. 本平台有权对评价内容进行审核、隐藏、删除或限制展示。

## 五、知识产权

1. 用户在本平台发布的评价内容，其著作权仍归用户所有。
2. 为提供本平台服务之目的，用户授权本平台展示、存储、整理其评价内容。
3. 本平台的源代码（除用户评价外的其他内容）受著作权法保护，未经许可不得复制、修改或分发。

## 六、隐私保护

1. 用户发布评价时可选择"匿名展示"，选择匿名后评价列表中不显示用户邮箱。
2. 无论是否选择匿名，评价对应的用户身份始终记录在数据库中，供平台管理和违规追溯使用。
3. 本平台不会向第三方出售、出租或共享用户的个人信息。
4. 在法律要求或为保护本平台合法权益的必要时，本平台保留向执法机关披露相关信息的权利。

## 七、浏览器插件说明
1. 浏览器插件仅在用户主动安装并启用后运行，不会在未经用户同意的情况下安装或运行。
2. 浏览器插件仅在南京大学教务系统等适配页面读取课程信息，用于查询和展示本平台的课程评价，不会修改教务系统原有业务逻辑或用户数据。
3. 浏览器插件不会收集与课程评价功能无关的网页内容，不会读取用户浏览其他网站的信息。
4. 浏览器插件与本平台后端通信时，仅发送实现评价查询、展示及发布所必需的数据。
5. 用户可随时通过浏览器扩展管理页面禁用或卸载浏览器插件。
6. 更多详细说明，请前往插件下载页面（nanping.eznju.com/download）查看。

## 八、免责声明

1. 本平台的评价内容由用户自行生成，不代表本平台立场，本平台不对评价内容的真实性和准确性作任何保证。
2. 本平台在收到权利人投诉或发现涉嫌侵权内容后，有权对相关内容进行删除、屏蔽或限制展示。
3. 课程信息（课程名、教师、开课时间等）来源于教务系统，可能存在延迟或不准确的情况，请以官方系统为准。
4. 本平台不对因服务中断、数据丢失、信息不准确等原因导致的任何直接或间接损失承担责任。
5. 用户因违反本协议或法律法规而产生的法律后果，由用户自行承担。

## 九、违规处理

1. 如用户发布的内容涉嫌违反本协议，本平台有权对该内容进行删除或隐藏处理。
2. 对于严重或多次违规的用户，本平台有权永久封禁其账号。
3. 用户如对处理决定有异议，可通过平台公告中的联系方式提出申诉。

## 十、协议修改

1. 本平台有权根据需要修改本协议，修改后的协议将在平台公布。
2. 协议修改后，用户继续使用本平台服务即视为同意修改后的协议。
3. 如用户不同意修改后的协议，应停止使用本平台服务。

## 十一、联系我们

如对本协议有任何疑问或建议，请通过平台公布的联系方式联系开发者。
