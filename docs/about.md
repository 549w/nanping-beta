# 关于「南评」

## 「南评」是什么？

南评（Nanping）是一个面向南京大学学生的第三方课程评价平台 + 浏览器插件。

「南评」平台及「Nanping」插件为个人业余开发维护的非营利项目，与南京大学无关。

## 浏览器插件

我们提供了一款浏览器插件，安装后可以在官方选课页面中直接查看课程的评分和评价，无需离开选课系统。插件不会收集个人信息，不会修改教务系统的任何功能。

详见[下载与安装教程](/download)。

## 技术栈

如果你对技术细节感兴趣：

- 后端：Python FastAPI + SQLAlchemy + SQLite，部署在 Docker 容器中
- 前端：原生 HTML + Pico.css + Vanilla JS，无框架依赖
- 浏览器插件：Chrome Extension Manifest V3，纯原生开发

## 数据来源

「南评」的课程信息来源于南京大学教务系统公开发布的课表数据。每当新学期的课表发布，作者都会进行一次数据导入。

「南评」的评价内容来自两部分：

- **历史评价**：从多个来源收集的历年课程评价，经过清洗和规范化后导入，标记为「系统导入」。值得一提的是，这部分评价的评分数据是由 LLM 根据评论内容生成的，仅供参考。
- **用户贡献**：注册用户通过本平台提交的原创评价。
