浏览器插件

安装 Nanping 浏览器插件后，在南京大学选课系统中即可一键查看课程评分和评价。

下载

download 下载浏览器插件（ZIP）

下载后请解压，按照下方教程安装即可。

安装教程

Chrome / Edge 浏览器

  1. 下载插件压缩包并解压到本地文件夹。
  2. 打开浏览器，在地址栏输入 chrome://extensions（Edge 为 edge://extensions）并回车。
  3. 打开右上角的「开发者模式」开关。
  4. 点击左上角「加载已解压的扩展程序」。
  5. 选择解压后的插件文件夹，确认即可完成安装。

提示：安装后建议将插件图标固定在浏览器工具栏，方便随时查看。

使用方法

  1. 登录南京大学选课系统。
  2. 进入选课页面后，插件会自动在每门课程旁显示评分徽章。
  3. 悬停在评分上可预览最新评价摘要。
  4. 点击评分可跳转到南评网站查看完整评价和撰写评价。

常见问题

插件不显示评分？

请确认：1）已正确加载插件；2）当前页面为选课系统课程列表页；3）网络连接正常。如仍无法使用，请加入 QQ 群反馈。

如何更新插件？

前往 GitHub Releases 下载最新版本，覆盖原有文件夹，然后在 chrome://extensions 中点击插件右下角的刷新按钮。

支持哪些浏览器？

所有支持 Chrome Extension Manifest V3 的浏览器均可使用，包括 Chrome、Edge、Arc、Brave 等。
